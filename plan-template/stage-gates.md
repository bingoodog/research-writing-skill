# 阶段门禁（Stage Gates）

> 用途：把写作全流程拆成阶段，避免跳步导致返工。

## 项目

- 项目名称：
- 文档类型（论文 / 项目文档）：
- 最后更新：
- 当前阶段：

## S0 启动与边界

- [ ] 已明确文档类型和目标
- [ ] 已明确输出物
- [ ] 已明确截止日期
- [ ] 已记录写作偏好（格式要求、列表规则等）

## S1 方案与范围确认

- [ ] 已确认文档范围和章节结构
- [ ] **[论文]** 已判断是否需要预注册
- [ ] **[项目文档]** 已确认目标读者和约束条件
- [ ] 已记录到 plan/project-overview.md

## S2 素材与资料准备

- [ ] **[论文]** 已记录文献检索式和检索日期
- [ ] **[论文]** 已完成初步去重和纳排
- [ ] **[项目文档]** 已收集项目背景材料和已有文档
- [ ] **[项目文档]** 已确认需求/设计的输入来源

## S3 大纲与方法

- [ ] 已完成章节骨架（plan/outline.md）
- [ ] 已对齐文档类型对应规范
- [ ] 已明确图表计划

## S4 写作与图表

- [ ] 已完成核心章节/节段初稿
- [ ] 已完成关键图表或脚本
- [ ] 已做一轮规范与质量检查

## S5 质量与合规

- [ ] 已做文档规范逐项核查
- [ ] **[论文]** 已做引用一致性检查
- [ ] **[论文]** 已检查伦理/AI使用披露要求
- [ ] **[项目文档]** 已做术语一致性检查
- [ ] **[项目文档]** 已做需求可验证性检查

## S6 交付包准备

- [ ] **[论文]** 已准备投稿材料（cover letter、作者声明等）
- [ ] **[论文]** 已完成期刊格式终检
- [ ] **[项目文档]** 已完成文档格式终检
- [ ] **[项目文档]** 已准备附件清单（代码、数据等）
- [ ] 已完成终稿版本管理

## S7 返修与归档

- [ ] **[论文]** 已建立审稿意见响应矩阵
- [ ] **[项目文档]** 已处理评审意见
- [ ] 已完成版本变更记录
- [ ] 已归档关键材料
